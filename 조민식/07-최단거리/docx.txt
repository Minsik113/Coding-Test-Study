😍최단거리 구하는 알고리즘
다익스트라, 플로이드 워셜

😀다익스트라 알고리즘의 동작 과정 -> O(ElogV)
1. 출발 노드를 설정
2. 최단 거리 테이블을 초기화
3. 방문하지 않은 노드 중에서 최단 거리가 가장 짧은 노드를 선택
4. 해당 노드를 거쳐 다른 노드로 가는 비용을 계산하여 최단 거리 테이블을 갱신
5. 위 과정에서 3번과 4번을 반복

😀플로이드 워셜 알고리즘 -> O(N^3)
모든 노드에서 다른 모든 노드까지의 최단 거리를 모두 계산한다.
2차원 테이블에 최단 거리 정보를 저장한다. DP유형에 속한다.
a->b까지의 거리랑 a->k->b인 거리랑 비교
Dab = min(Dab, Dak + Dkb)

노드 개수가 N개일 때, N번의 단계를 수행한다. 
- 각 단계마다 O(N^2)의 연산을 통해 현재 노드가 거쳐 가는 모든 경로를 고려한다.
-> O(N^3)

노드 개수가 500개 이하일 때 가능하나 500*500*500 = 1억이 넘음. 숫자 작을 때 사용
